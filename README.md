# Innovare Side Menu

A beautiful, highly customizable side menu widget for Flutter.

> **🚧 Under Development** — This package is being actively developed. Full documentation and examples will be available with the 1.0.0 release.

## Features

- Declarative menu with sections and hierarchical items
- Expanded and collapsed (rail) modes
- 5 built-in themes
- Badge support (count, dot, custom)
- Permission-based item filtering
- Full accessibility support
- Zero external dependencies

## Getting Started

```dart
import 'package:innovare_side_menu/innovare_side_menu.dart';
```
